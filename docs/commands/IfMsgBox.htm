IfMsgBox

检查用户在最近的 MsgBox 命令中点击了哪个按钮。

IfMsgBox, ButtonName

参数

ButtonName

下列字符串之一表示用户在最近的 MsgBox 命令中所点击的按钮:

是
No
OK
Cancel
Abort
Ignore
Retry
Continue [v1.0.44.08+]
TryAgain [v1.0.44.08+]
Timeout (即单词 "timeout" 表示 MsgBox 命令 超时)

相关

MsgBox

示例

MsgBox, 4, , Would you like to continue?, 5  ; 5 秒的超时时间.
IfMsgBox, No
    Return  ; 用户点击了 "No" 按钮.
IfMsgBox, Timeout
    Return ; 即在超时时假设点击了 "No".
; 否则, 继续:
